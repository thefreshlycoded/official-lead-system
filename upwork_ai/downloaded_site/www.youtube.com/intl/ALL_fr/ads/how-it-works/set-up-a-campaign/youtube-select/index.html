Conditions d'utilisation de cette offre :

Dans les conditions ci-dessous, "Google Ads" peut se rapporter à Google Ads ou Google Ads Express, selon le cas.

  1. Cette offre est destinée aux clients dont l'adresse de facturation est située aux États-Unis. Un seul code promotionnel est autorisé par annonceur.
  2. Pour activer cette offre : saisissez le code promotionnel dans votre compte avant le 31 décembre 2020. Pour bénéficier de cette offre, vous devez saisir le code dans un délai de 14 jours à compter de la première impression d'annonce diffusée via votre premier compte Google Ads.
  3. Pour bénéficier de l'avoir : dans les 30 jours suivant la saisie du code, vos campagnes publicitaires doivent générer des frais d'au moins 50 $ (hors taxes). Notez qu'un paiement de 50 $ n'est pas suffisant. Le suivi de vos frais publicitaires démarre une fois que vous avez saisi le code, jusqu'à ce qu'ils atteignent 50 $.
  4. Une fois les étapes 2 et 3 effectuées, l'avoir est généralement appliqué sur le récapitulatif de facturation de votre compte dans les cinq jours suivants.
  5. Les avoirs ne s'appliquent qu'aux frais publicitaires à venir et non à ceux qui ont été générés avant la saisie du code.
  6. Une fois que vous aurez dépensé votre avoir, vous ne recevrez pas de notification. Les frais publicitaires supplémentaires vous seront facturés selon votre mode de paiement. Si vous ne souhaitez plus diffuser de publicité, vous pouvez suspendre ou supprimer vos campagnes à tout moment.
  7. Pour prétendre à un avoir promotionnel, vous devez recevoir des factures Google Ads, et votre compte doit être en règle.
