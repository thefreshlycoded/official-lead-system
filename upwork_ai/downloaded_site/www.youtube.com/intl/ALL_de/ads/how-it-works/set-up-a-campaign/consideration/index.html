Nutzungsbedingungen für dieses Angebot:

Der Begriff „Google Ads“ kann sich im Folgenden auf Google Ads oder Google Ads Express beziehen.

  1. Das Angebot gilt nur für Kaufinteressierte mit Rechnungsadresse in den USA. Es ist auf einen Gutscheincode pro Werbetreibende*n beschränkt.
  2. Aktivierung dieses Angebots: Geben Sie den Gutscheincode bis zum 31. Dezember 2020 in Ihrem Konto ein. Um dieses Angebot in Anspruch zu nehmen, müssen Sie den Code innerhalb von 14 Tagen nach der ersten Anzeigenimpression für Ihr erstes Google Ads-Konto eingeben.
  3. So erhalten Sie die Gutschrift: Nach Eingabe des Codes müssen für Ihre Werbekampagnen innerhalb von 30 Tagen Kosten in Höhe von mindestens 50 $ netto anfallen. Eine Zahlung in Höhe von 50 $ ist nicht ausreichend. Sobald Sie den Code eingegeben haben, erfassen wir Ihre Werbekosten und ermitteln, ob Sie den Betrag in Höhe von 50 $ erreichen.
  4. Sobald die Schritte 2 und 3 abgeschlossen sind, wird die Gutschrift für gewöhnlich innerhalb von 5 Tagen in der Abrechnungsübersicht Ihres Kontos aufgeführt.
  5. Die Gutschrift kann ausschließlich für zukünftige Werbekosten verwendet werden und nicht für Kosten, die vor der Eingabe des Codes entstanden sind.
  6. Sie werden nicht benachrichtigt, wenn das Guthaben aufgebraucht ist. Alle weiteren Werbekosten werden über Ihre ausgewählte Zahlungsmethode abgerechnet. Wenn Sie keine Anzeigen mehr schalten möchten, können Sie Ihre Kampagnen jederzeit pausieren oder löschen.
  7. Sie erhalten das Startguthaben nur, wenn Ihr Konto bereits erfolgreich über Google Ads belastet wurde und eine einwandfreie Bonität aufweist.
